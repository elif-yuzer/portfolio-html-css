```
property: clamp(MINIMUM, TERCİH_EDİLEN, MAKSIMUM);
h1 {
  /* clamp( En Küçük Boyut, İdeal/Akışkan Boyut, En Büyük Boyut ) */
  font-size: clamp(1.5rem, 4vw + 1rem, 4rem);
}
```
clam fonksiyonu sayesinde media query e gerek kalmıyor.
```css
/* FONT BOYUTU */
font-size: clamp(16px, 14px + 0.5vw, 24px);
```

**Ne olur?**

- Mobilde (320px): 16px (minimum)
- Tablette (768px): ~18px (14 + 768'in %0.5'i)
- Desktopda (1920px): 24px (maksimum)
- ```css
/* PADDING */
padding: clamp(20px, 10px + 2vw, 80px);
```

**Ne olur?**

- Mobilde (320px): 20px
- Tablette (768px): ~25px (10 + 768'in %2'si)
- Desktopda (1920px): ~48px

```css
/* GENİŞLİK */
width: clamp(300px, 50vw, 1200px);
```

**Ne olur?**

- Her zaman ekranın %50'si kadar genişlikte
- Ama en az 300px, en fazla 1200px
